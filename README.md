# Composite

## 📌 Descripción
Este proyecto representa un ejemplo de la aplicación del patrón **Composite** junto con el patrón **Builder y Prototype** en el desarrollo de software.

El objetivo es demostrar cómo estos patrones de diseño pueden trabajar en conjunto para mejorar la flexibilidad, reutilización y organización del código.

Consiste en un Sistema de Archivos donde muestra Proyectos JAVA o WEB, nombres de los archivos y su tamaño en bytes.
